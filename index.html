Meus Projetos

Sistema de Recomendação de Filmes Personalizado

Desenvolvi um sistema de recomendação de filmes usando React no front-end e Node.js no back-end, com MongoDB para armazenamento de dados. A aplicação sugere filmes com base no histórico do usuário, integrando a API do TMDB para obter informações sobre os filmes. Utilizei JWT para autenticação e fiz o deploy no Heroku e Netlify. O projeto me proporcionou experiência em desenvolvimento full-stack e integração de APIs.

Gerenciador de Tarefas

Desenvolvi uma plataforma de gerenciamento de tarefas colaborativas usando React no front-end e Express no back-end, com PostgreSQL para armazenamento de dados. A aplicação permite que usuários criem, editem e compartilhem tarefas em tempo real, com funcionalidades de filtragem e categorização. Utilizei WebSockets para comunicação em tempo real e OAuth2 para autenticação segura via Google. O deploy foi feito no Heroku, garantindo escalabilidade e performance. Esse projeto me proporcionou uma excelente experiência em desenvolvimento full-stack, manipulação de dados em tempo real e integração de sistemas de autenticação.
